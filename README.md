## Currency Converter

A simple and responsive **Currency Converter web app** built using  
**HTML, CSS, and JavaScript**.  
It converts currencies using real-time exchange rates.

##  Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Exchange Rate API


##  Features
- Real-time currency conversion
- Clean and responsive UI
- Easy to use
- Beginner-friendly project
